Rust Raid Calculator

This application is a simple way to estimate the amount of explosives needed to raid a base size that you input for the lowest cost of sulfur. Note: With any wooden object, it is better to use either a flamethrower or molotovs, and while rockets are generally more expensive per sulfur, they are capable of splash damage, so use rockets when you want widespread damage i.e splash four floors instead of one.
